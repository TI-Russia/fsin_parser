# fsin_parser
Парсер для обработки файлов Федеральной службы исполнения наказаний 2015 и последующих годов.
Используется open-source библиотека xlnt для доступа к excel файлам ( https://github.com/tfussell/xlnt )
# Сборка
Для сборки необходимы Visual Studio 2015, Qt Qt5.8.0 сборка под Visual Studio 2015 x64, CMake версии 3.7 и выше

В файле source/CMakeList.txt нужно прописать правильный путь к Qt

Сборка запускается с командрной строки файлом build/build.bat

Исполняемый файл запускается с двумя параметрами - имя exсel файла и имя закладки, в которой содержатся данные.

Например
    fsin_parser.exe fsin_2015.xlsx 2015
    fsin_parser.exe fsin_2016.xlsx 2016
